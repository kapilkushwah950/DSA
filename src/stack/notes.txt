stack
-------
1.it is a linear ds.
2.it follows last in first out principle [LIFO]last in first out.
3.operation
  PUSh - insert an elemant in the stack
  pop -  remove/delete a  elements from stack
  peek - top most value in the stack
4. all opertion of stack data structure work on the basis of top.
5.condition
  a. TOP ==-1,stack is empty
  b. TOP == 0 ,single elemant in the stack
  c. TOP == size-1,stack is full
  d. TOP == size, stack overflow
